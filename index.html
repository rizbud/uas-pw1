Pertemuan 1: Pengenalan Dasar HTML

Pada kesempatan kali ini, kita akan belajar mengenai dasar-dasar HTML. HTML adalah singkatan dari HyperText Markup Language. HTML adalah bahasa markup standar untuk membuat halaman web. HTML menggambarkan struktur konten sebuah halaman web secara logis. HTML menggunakan elemen-elemen dasar untuk memformat konten seperti heading, paragraph, list, link, dan lain-lain.
